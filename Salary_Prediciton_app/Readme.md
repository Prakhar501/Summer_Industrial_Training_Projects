# 💼 Salary Prediction System

A simple end-to-end Machine Learning application that predicts an employee's salary based on years of experience using **Linear Regression**. The model is deployed as an interactive web application using **Streamlit**, allowing users to make real-time predictions.

> This project was developed as part of my **Summer Industrial Training at DUCAT**, where I explored the complete Machine Learning pipeline—from data preprocessing and model training to deployment.

---

## 🚀 Features

- Predicts salary based on years of experience
- Interactive web interface built with Streamlit
- Linear Regression model trained using Scikit-learn
- Simple, lightweight, and easy to use
- Beginner-friendly end-to-end Machine Learning project

---

## 📈 Model Used

**Linear Regression**

Linear Regression is a supervised Machine Learning algorithm used to predict continuous values by learning the relationship between input and output variables.

For this project:

- **Input:** Years of Experience
- **Output:** Predicted Salary

## ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/<your-username>/Industrial_Training_Projects.git
```

### Navigate to the project

```bash
cd Salary_Prediction_System
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the Streamlit application

```bash
streamlit run app.py
```

The application will open in your browser at:

```text
http://localhost:8501
```
